程式目的:
從 Kaggle 下載 70 Dog 資料集，對資料集中的15個指定類別做深度學習之訓練、分類、測試等練習。


實作方式:
使用keras的方法建立模型，並選擇卷積神經網路(Convolutional Neural Network, CNN)的模型進行訓練。於程式中建立了一個 DogClassifier 類別執行這次的模型訓練，以下分別介紹此類別中，各個方法的功能:
__init__() : 設定初始值。

build_model() : 建立模型，這次選擇的是卷積神經網路(CNN)的模型。

train() : 從訓練集以及驗證集抓取訓練樣本，提供給模型訓練。訓練樣本為15種狗類的影像，大小為224 * 224。

evaluate() : 訓練過後的模型，針對驗證集做測試，並將 Accuracy 輸出至 validation_accuracy.xlsx。

center_crop_image() : 回傳以中心為準，挑整過後的影像。

get_predicted_breed() : 回傳預測的狗類品種。

test() : 讀取實驗室提供的300張測試影像；因為每張影像的大小都不一樣，所以呼叫 center_crop_image() 以及 resize() 對影像作預先處理，處理方式為以影像中心為準裁切出224 * 224的大小；接著將處理過後的影像樣本傳遞給 get_predicted_breed() 預測狗類品種；最終輸出至Excel檔案。


心得感想:
因為是第一次接觸深度學習的領域，因此花了很多時間在學習上，在實作上要怎麼把程式封裝的比較好用也是十分困難。這次作業最大的考驗我認為是訓練集的影像大小與實驗室真正要測試的影像大小不一樣，因此我的解決方法是先用肉眼觀察，發現300張測試影像的狗狗大多都出現在影像中心，所以我讓測試影像預先接受了處理，將他們以中心為準裁切成適當的大小，再交給模型測試，期望能得到較高的效能。
